The World Wide Web

The World Wide Web(simply called 'www') is the collection of HTML that bases for almost every page. Plain text, HTML, Images, Videos, Music, etc... can be found on the web.
There are some components of the web. I just explained them below.

Browser : Browser such as Chrome, IE, FireFox makes requests via the internet to servers.

HTTP : This is a protocol that browser uses to communicate with servers.

INTERNET : The most largest computer network.

Servers : Hosts the files.

HTML

The web is made with HTML(hyper text markup language) that consist of text content, markup, references to other documents, other links.

Text content : Plain text is just plain text in html

Markup : This made up with tags. Tags are consisted of opening tags and closing tags and the contents which is in between openiing,closing tags. We call whole these as elements. There are some examples about tags.

- bold tag : The content that locate between this tag will be bolded.
- emphasis tag : The content that locate between this tag will be emphasized.
- paragraph tag : The context that locate between this tag will be covered with invinsible block. This is a block element.
- break tag : This ends a line. So, the text that comes out after this tag, it will be in just below. And this is a inline element.
- span tag : This is a inline element and some attributes that can change content style might be used(by using class).
- div tag : This is a block element and some attributes that can change content style might be used(by using class). This also makes invisible box like paragraph tag.
- image tag : This is for including images. It has some attributes called src and alt. src indicates the image url and if there's no images in it, alt attributes will be used.
- anchor tag : This one is for making links. It has href attribute that incidates URL. If I use this tag to link to Udacity site, it will show up like this. (Go to Udacity).

and the last but not least, all the white space, new lines, tabs will be rendered as one single space. If you want to display texts in new line. You should use break tag.

HTML Documents

DOCTYPE : This is what kind of HTML is.

html tag : This surrounds the entire document.

head tag : This has kind of metadata and other random stuff such as javascript, css, title tag.

body tag : Actual contents of the document.
